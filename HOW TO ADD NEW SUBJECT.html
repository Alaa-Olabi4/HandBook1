طريقة إضافة موضوع جديد

  1. يتم إنشاء ملف JavaScript (لاحقته .js) باسم الموضوع ضمن التصنيف المطلوب إضافته.
  2. يجب عدم وجود مسافات ضمن اسم الملف.
  3. يتم تعبئة الحقول كما هو مذكور ضمن الملف template.js.
  4. يتم إضافة الموضوع الجديد ضمن ملف index.js الموجود في مجلد التصنيف الذي يتم الإضافة له.
  5. بأعلى الملف يتم إضافة:
    import superclip from './superclip.js';
  6. ثم ضمن المتغير articles يتم إضافة اسم الملف الجديد.
